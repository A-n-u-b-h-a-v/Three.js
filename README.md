## Projet 1 - Cyberpunk

**Features**

- **3D Rendering**: Utilizes Three.js for rendering a dynamic 3D scene.
- **Post-Processing Effects**: Implements RGB shift and other effects using EffectComposer and ShaderPass.
- **Interactive Mouse Controls**: Allows scene rotation based on mouse movement.
- **Responsive Design**: Automatically adjusts rendering size on window resize.
- **HDR Environment Mapping**: Loads an HDRI for realistic lighting and reflections.
- **GLTF Model Loading**: Supports loading and displaying 3D models in GLTF format.

## Project 2 - Planets

**Features**

- **3D Planetary Simulation**: Visualize four distinct planets with realistic textures.
- **Starry Background**: A high-resolution starry sky enhances the scene's aesthetics.
- **Interactive Rotation**: Rotate planets using the mouse wheel for an engaging experience.
- **Smooth Animations**: GSAP is used for fluid transitions and animations.
- **Responsive Design**: Adapts to various screen sizes for consistent user experience.
- **HDRI Lighting**: Realistic lighting effects using HDR images.
- **Orbit Controls**: Easy navigation around the 3D environment.
